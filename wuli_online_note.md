# 矢量分析

梯度：标量（势函数）/ 向量（方向向量）= 矢量场（梯度）
散度：向量（密度函数）/ 向量 = 标量（通量）
拉普拉斯：梯度的散度
旋度：向量（矢量算符）X 向量（矢量场） = 向量（旋度）
调和场：调和场的充分必要条件是存在拉普拉斯为零的势函数
	- 旋度为零：积分与路径无关， 必定可以定义势函数
	- 散度为零：拉普拉斯为零
